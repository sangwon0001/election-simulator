# 쌍둥이 투표동 시뮬레이터

[![DOI](https://zenodo.org/badge/1264874211.svg)](https://doi.org/10.5281/zenodo.20624234)
논문: [국문](paper/twin_precincts_kr.md) · [English](paper/twin_precincts_en.md) · [PDF](paper/twin_precincts_en.pdf)

제9회 전국동시지방선거(2026-06-03) 시·도지사선거에서, 서로 다른 **두 개표단위(읍면동×구분)**가
A·B 두 후보 득표수가 **동시에 똑같은** '쌍둥이 투표동'이 발생할 확률을 추정한다.
실제 중앙선관위 데이터를 크롤링해 경험적 관측과 몬테카를로 시뮬레이션을 비교한다.

> 쌍둥이 = 투표소(투표구)가 아니라 **개표단위(동×구분)** 기준. 두 단위 i, j에서
> `A_i == A_j` **그리고** `B_i == B_j` 인 충돌(생일 문제 구조).

## 방법론

- **단위**: 읍면동 × 구분(관내사전투표 / 선거일투표). 거소·관외사전·합계는 제외.
- **A, B 후보**: 시·도별 총득표 상위 2명 자동 식별 (전남은 민형배/이정현).
- **캐노니컬 모델 (`twin_model.py`)**: 동별 **전체유권자 N**에서
  `N ──multinomial──> 사전투표자 / 본투표자 / 기권` (동별 사전·본 투표율) →
  각각 동별 지지율로 득표 갈림(이항). 모든 모수가 동별 실측값, 커스텀 파라미터 0개.
  랜덤성 = 누가 사전/본/기권하나(multinomial) + 표가 어떻게 갈리나(이항).
- **충돌 탐지**: `(A,B)`를 정수키 `A·BIG+B`로 인코딩 → 정렬 → 중복 카운트. O(M log M).
- 모델을 어떻게 정교화해도(V고정 / V랜덤 / 동별실제 / multinomial) P(≥4)≈3.5%로 수렴.

## 핵심 결과

### 전남도지사 (A=민형배 80%, B=이정현 12%)

- **실제 관측: 동 단위 0쌍, 동×구분 4쌍** (모두 관내사전투표):
  여수 삼일동=신안 하의면, 장성 북하면=함평 엄다면, 보성 노동면=신안 팔금면, 화순 이양면=강진 병영면
- **기본 모델 P(4쌍 이상) ≈ 2.5%**, 기대 ~1.1쌍 → 4쌍은 상위 꼬리의 "운 따른 해".

### 17개 시·도 비교 — "압승만으론 안 되고, 농촌 소규모가 필수"

| 시·도 | A점유율 | 실제 쌍 | 시뮬 기대 |
|---|---|---|---|
| **전남** | 77% | **4** | 1.20 |
| 경북 | 67% | 1 | 0.50 |
| 전북 | 50% | 1 | 0.45 |
| 경남 | 51% | 1 | 0.28 |
| **광주** | **76%** | **0** | 0.02 |
| 경기·서울·기타 | ~50% | 0 | <0.13 |

- **광주는 전남과 같은 압승(76%)인데 쌍둥이 0쌍** — 도시라 동이 크고 득표수가 넓게 퍼짐.
- 전남이 유일한 outlier인 이유 = **3중 조합**:
  ① 압승(B 득표가 수십 표에 뭉침) ② 농촌 소규모 면(작은 단위 多)
  ③ 극단적 사전투표 쏠림(관내사전 51%, 전국 유일 >50% → 선거일 단위까지 잘게 쪼개짐)

## 엔진 메모 (`twin_vote_sim.py`)

- 단위별 `turnout_mean_per_unit`, `support_per_unit`, `alpha_per_unit` 배열 지원.
- `dispersion`(φ): sub-binomial(φ<1) 정규근사. 동전던지기보다 변동 작은 실제 투표 모사용.
- `SimResult.p_exactly(k)`, `p_at_least(k)`, `pair_distribution()`: 쌍 개수 확률.
- 다항분포는 이항분해(`X_A~Bin(V,pA)`, `X_B~Bin(V-X_A, pB/(1-pA))`)로 완전 벡터화.

## 진단으로 밝힌 모델의 한계

근접쌍 분포(`diagnose.py`, 전국 정식판은 `analyze_national.py`)를 보면 bulk(거리 d≥2)는
실제와 일치하나, **정확일치(d=0)만 실제가 모델의 ~2.8배**. 전국 기준 관측 9쌍 vs 기대
3.23(캐노니컬) / 3.47±0.18(모형무관 사이드밴드 외삽) → **P(≥9) = 0.64% / 0.94%**.
초과는 근접이 아니라 정확일치에만 있는 원자(atom)라서 매끄러운 메커니즘으론 양쪽
방향(자연 군집으로 설명 / 모델 비판으로 해소) 모두 못 만든다. 상세 논증은 `FINDINGS.md`.

## 결론

같은 답(~1%)이 독립인 다섯 경로(조건부 캐노니컬 / V변형 / 모집단추출 / ab-initio
분포기반 / 모형무관 사이드밴드)에서 나왔고, 제21대 대선 반복 검증에서는 초과가
재현되지 않았다(관측 7 vs 기대 6.0, P=40%).

> 두 동네 득표수가 정확히 겹치는 건 매 선거 몇 쌍씩 생기는 평범한 현상이고
> (대선에서도 7쌍 — 모델 예측과 일치), 지선 광주전남의 5쌍은 그중 드문 편 —
> 약 100번의 선거에 1번, 로또 1등(1/814만)보다 8만 배 흔한 우연이다.
> 시뮬레이션 없이 데이터의 근접쌍을 직접 세어봐도, 관측값을 버리고 분포만 주고
> 가상 지역을 만들어봐도 같은 답이 나온다. 신기한 일은 맞다. 증거는 아니다.
> 그리고 신기한 일은 매 선거 어디선가 반드시 하나씩 나온다.
