# 쌍둥이 투표동 연구 정리 (2026-06-10)

제9회 전국동시지방선거 시·도지사선거 관내사전투표/선거일투표에서,
서로 다른 두 개표단위(읍면동×구분)의 두 주요후보 득표가 **정확히 일치**하는
'쌍둥이'가 통계적으로 의심스러운가? — 를 실제 데이터 + 몬테카를로로 검토.

발단: 통계학자 허명회 교수가 "인천 1쌍, 광주전남 5쌍은 자연스러운 우연"이라 분석.
우리는 실제 데이터를 직접 크롤링해 더 정밀하게 검증함.

---

## 1. 데이터

- **출처**: 중앙선관위 선거통계시스템(info.nec.go.kr), 개표단위별 개표결과.
- **수집**: Playwright 자동 크롤러(`src/crawl_nec.py`)로 17개 시·도 전체.
  - `data/crawl_provinces/<코드>_<시도>.json` (17개)
  - 전남은 수동 크롤 22개 시군구도 별도 보관(`data/jeonnam_manual/`)
- **단위**: 읍면동 × 구분(관내사전투표 / 선거일투표). 거소·관외사전·합계 제외.
- **A, B 후보**: 시·도별 총득표 상위 2명 자동 식별 (전남/광주는 민형배/이정현).
- **데이터 구조(중요)**: 동의 `계 선거인수 = 관내사전 선거인수 + 선거일 선거인수`.
  관내사전 선거인수 ≈ 투표수(카테고리가 "사전투표함"으로 정의됨).
  선거일 선거인수 − 기권 = 선거일 투표수. 거소·관외사전은 동 단위로 안 쪼개짐.

## 2. 캐노니컬 모델 (`src/twin_model.py`)

동별 전체유권자 N에서 multinomial로 분배 후 후보 갈림 (모든 모수 동별 실측):

```
N(선거인수) ──Binomial──> Ve(사전투표자) = N × 사전투표율
            ──Binomial──> Vb(본투표자)   = (N−Ve) × 본투표율(조건부)
Ae = Binomial(Ve, 사전 민형배지지율)   # 사전 단위
Ab = Binomial(Vb, 본  민형배지지율)   # 본  단위
→ 전체 단위 중 (A,B) 동시 일치 쌍 카운트
```

랜덤성: ① 누가 사전/본/기권하나(turnout) ② 표가 어떻게 갈리나(split). 둘 다 이항(φ=1).

## 3. 핵심 결과

| 구분 | 실제 쌍둥이 | 모델 기대 | P(관측 이상) |
|---|---|---|---|
| 전남 단독 | 4 | 1.2 | ~3.5% |
| 광주+전남 합본 | 5 (전남4+교차1) | 1.5 | ~1.7% |
| **전국(광주전남 합본 포함)** | **9** | **3.23** | **0.64%** |
| **전국 — 사이드밴드(모형무관)** | **9** | **3.47±0.18** | **0.94% [95%: 0.49~1.66%]** |

- 전국 9쌍 = 광주전남 5(교차1 포함) + 경북·경남·전북·인천 각 1.
  광주·전남은 같은 선거구(양대후보 동일)라 **한 블록으로 합본**해 교차쌍 포함 집계.
- 교차 쌍둥이: 고흥 금산면 = 광산 송정1동 (민형배 1401, 이정현 120, 둘 다 사전).
- **유형 분해 (이전 기록 "9쌍 전부 관내사전"은 오류였음 — 데이터로 정정)**:
  관내·관내 **6쌍**(인천 송도1동=송도2동, 전남 4, 광주전남 교차 1)
  + 선거일·선거일 **2쌍**(경북 의성 점곡면=영덕 창수면, 경남 하동 양보면=거창 위천면)
  + 관내·선거일 혼합 **1쌍**(전북 익산 웅포면=진안 성수면).
- 쌍둥이 발생 조건 = **압승/소규모 면(B 득표가 42~180표에 뭉침)**. 사전투표 쏠림(관내사전
  51%, 전국 유일 >50%)은 전남 한정 가속 요인. 광주는 압승이나 도시라 0.

## 4. 모델 강건성 (여러 각도로 다 ~같은 답)

전국 P(≥9)가 모델 선택에 거의 무관함을 확인:

| 변형 | 결과 |
|---|---|
| V 고정 vs V 랜덤(turnout) | 거의 동일 (~1.2, 전남) |
| split 분산 φ = 1 → 0.3 | P 0.59% → 0.80% (거의 평평) |
| 관측지지율 vs 모집단추출 | 1.46 → 1.37 (오히려 더 드묾) |
| 유한모집단 보정 φ≈0.78 | 0.72% (이항과 동일) |

→ **전국 ~0.7%는 강건.** 어떻게 흔들어도 안 바뀜.

## 4b. 플러그인 편향 검증 + 모형무관 사이드밴드 (2차 세션 — 결정판)

§6의 "관측모수 + 이항노이즈 부트스트랩은 매끄러움 가정을 공유한다"는 자기비판을
정면으로 검증했다. 결과: **반론은 이론적으로 실재하나 이 데이터에선 효과 없음.
0.7%대 꼬리는 모형 없이 데이터만으로 재확인됨.**

**(1) 플러그인 노이즈 이중계산 — 이론 + 해석적 증명 (`src/bias_demo.py`)**
관측값 p̂=A/V 중심 재추첨은 두 단위의 시뮬 득표차를 (관측차)+(새 노이즈)로 만드는데,
관측차에 이미 같은 크기의 노이즈가 들어 있어 차이분포가 노이즈 두 번 컨볼브가 된다.
→ 진짜 평균이 같은 쌍의 정확일치 확률을 1차원당 1/√2, (A,B) 2차원 ~1/2로 과소평가.
이항 pmf 내적으로 정확 계산: 측정 0.707 = 이론 1/√2 (제각각 쌍은 0.985≈1, 편향 없음).

**(2) 그러나 실데이터에선 편향이 안 보임 — 근접쌍 셸 r(d) 진단**
전국 셸 카운트 n(d) (체비셰프 거리, d=1..15) 의 관측/시뮬 비율 r(d):
d≥2 에서 r≈0.88~1.27로 평탄(≈1) — **모델이 근접쌍 밀도를 정확히 재현**한다.
만약 진짜동일 쌍 모집단이 유의미했다면 r(d)가 전 구간 ~2로 떴어야 함. 즉 편향의
전제(숨은 동일평균 쌍 다수)가 데이터에 없다. **튀는 건 오직 d=0 (r=2.78) 하나.**

**(3) 사이드밴드(bump-hunt) 외삽 — 이항·φ·turnout 가정 전부 불필요**
ρ(d)=n(d)/(8d) (L∞ 셸 격자점 정규화, 비순서쌍 일관성은 합성 테스트로 검증)를
d∈[1,15]에서 Poisson-ML 적합 후 d=0 외삽:
선형 ρ̂(0)=3.47±0.18 → P(≥9)=0.94% [0.49~1.66%] / 2차 3.64±0.36 → 1.26% [0.33~3.39%].
자가검증: 시뮬 데이터에 같은 외삽 적용 시 시뮬 d=0 (3.24 vs 3.27) 정확 복원.

**(4) 의미** — 캐노니컬 0.64%와 사이드밴드 0.94%가 수렴. "모델을 잘못 세워서 나온
숫자"라는 류의 반론은 종결. 초과분은 근접(d=1~15)이 아니라 **정확일치(d=0)에만 있는
원자(atom)** — 매끄러운 자연 메커니즘(인구 동질성 등)은 "거의 같음"(d=1~수십)도 함께
늘려야 하는데 그 흔적이 없다. 남는 설명은 ① ~1% 포아송 우연 ② 비통계적 원인
(데이터 처리·보고 단계 인공물 포함) 둘뿐이며, 단일 선거로는 식별 불가(§6).

## 4c. 반복 검증 — 제21대 대선(2025-06-03): 초과 재현 안 됨

같은 방법론을 대선에 적용 (`src/analyze_president.py`, 공공데이터포털 CSV
`data/president_20250603.csv`). 대선은 전국이 같은 후보(A=이재명, B=김문수)라
**전국 한 블록**: 동 3,554 → 단위 7,108, 시·도 경계 없이 전 쌍 비교.

| | 관측 | 캐노니컬 기대 | P(≥관측) | 사이드밴드 ρ̂(0) | P(≥관측) |
|---|---|---|---|---|---|
| 대선 전국 | **7** | 6.02 | **39.7%** | 6.44±0.24 (선형) | 46.3% |

- r(d) = 0.93~1.17 **전 구간 평탄 (d=0 포함)** — 원자 없음. 모델·셸·관측 3자 일치.
- 검정력 비교: 대선은 기대 자체가 커서(6.0), 지선과 같은 ×2.8 초과 메커니즘이
  있었다면 ~17쌍이 나왔어야 함 (P(≥17|6)≈0.01%). 관측 7 → 그 메커니즘 부재를 강하게 배제.
- **유형 역전**: 지선 관내6·선거2·혼합1 ↔ 대선 **사전·사전 0**·선거6·혼합1.
  쌍둥이는 사전투표가 아니라 "작은 득표수가 사는 곳"을 따라다님 — 지선 전남은 사전
  쏠림(51%)으로 사전 단위가 작았고, 대선은 농촌 면 선거일 단위가 작음.
  "사전투표 특이" 서사와 정면 배치되는 구조적 증거.
- 7쌍 위치: 호남 면 중심 + 강원·경남 교차 + 경기·충북 — 압승/소규모 조건은 동일.

**의미**: 지선 9쌍(0.6~0.9%)의 정체에 대한 첫 식별 증거. 더 검정력 높은 대선 반복에서
초과가 완전히 사라짐 → "선거 전반에 반복되는 구조적 원자" 가설은 기각. 지선 결과는
사후선택된(허명회 논쟁이 발단) **일회성 ~1% 요동**이거나 지선 한정 원인 — 후자는
미배제이나 현재 무근거. 방법론 자체는 대선에서 3자 일치로 검증 완료.

## 5. 폐기된 가설들 (중요 — 다음 세션에 반복 금지)

- ❌ **"사전투표 100% turnout → 유한모집단 → sub-binomial"**: 데이터 오독.
  관내사전 선거인수는 "사전투표한 사람 수"라 turnout 100%는 동어반복. 분산 줄일 근거 없음.
- ❌ **"공통 지지율 군집으로 자연스럽다"**: 분산분해 결과 동별 지지율 차이의
  **93%가 진짜 차이**, 7%만 표본노이즈. 공통 가정은 데이터가 기각.
- ❌ **sub-binomial(φ<1)로 5쌍 재현**: (a) 캘리브 0 (b) 단일선거로 within-dong 분산
  측정 불가 (c) split φ 낮춰도 turnout 랜덤이 지배해 효과 거의 없음. 근거 없는 손잡이.
- ❌ **"플러그인(관측모수 재추첨)이 기대 쌍둥이를 과소평가 → P값 과장"**: 이론적으로는
  진짜(1/√2 편향, `bias_demo.py`로 해석적 증명)지만, 셸 진단 결과 r(d≥2)≈1 — 편향의
  전제(동일평균 쌍 모집단)가 이 데이터에 없음. 사이드밴드가 캐노니컬 기대를 재확인(§4b).

## 6. 근본적 한계

- **식별 불가능**: 쌍둥이의 "초과분"(관측 9 vs 기대 ~3.2-3.5)의 정체를 이 데이터로 판정
  불가. 단, §4b로 한 단계 좁혀짐 — 초과는 d=0 원자라서 "매끄러운 양성 군집(인구 동질성)"
  설명은 어려워졌고, 남는 후보는 ~1% 우연 vs 비통계적 원인.
- **단일 선거**: 각 동이 1회 관측이라 within-dong 분산(φ) 측정 자체가 불가능.
- **방법론 성격**: ~~두 방법 다 이항·매끄러움 가정 공유~~ → §4b 사이드밴드로 해소.
  기대치 추정은 이제 모형무관 경로(셸 외삽)로도 같은 답이 나옴.

## 7. 정직한 결론

- 전국 9쌍은 캐노니컬 모델 **0.64%**, 모형무관 사이드밴드 **0.94% [0.49~1.66%]**
  — 약 100~150번에 1번. 드문 편이나 불가능 아님. **모델 선택의 문제가 아니라
  데이터 자체의 성질**임이 확정됨(§4b).
- 초과가 **전국 골고루** 분포(전남5·경북·경남·전북·인천) + 유형도 사전6·본2·혼합1로
  섞임 → 특정지역·특정투표유형 조작 시나리오와 안 맞음.
- 초과는 **정확일치(d=0)에만 있는 원자** — 근접쌍(d=1~15)은 모델과 일치. 매끄러운
  양성 설명(인구 동질성)도, 매끄러운 모델 비판도 둘 다 이 패턴을 못 만든다.
- **부정의 증거 아님. 허명회 교수의 "지극히 자연스러움"보다는 드문 tail 사건.**
- **대선 반복 검증(§4c)에서 초과 미재현** (관측7 vs 기대6.0, P=40%, r(d) 전 구간 평탄)
  → 반복되는 구조적 메커니즘은 기각. 지선 9쌍은 사후선택된 일회성 ~1% 요동이
  가장 유력하고, 지선 한정 원인은 미배제·무근거. 유형 역전(지선 사전6/9 ↔ 대선 사전0/7)은
  "쌍둥이는 사전투표가 아니라 작은 득표수를 따라다닌다"를 확정 — 사전투표 서사 배제.
